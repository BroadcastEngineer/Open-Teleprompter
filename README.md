# Open-Teleprompter
Demo: https://github.com/BroadcastVision/open-teleprompter

# Control Keys
* Increase font size [+]
* Decrease font size [-]
* Text Scroll [UP or DOWN Works only when the prompter in on Stop mode]
* Decrease Scroll Speed [LEFT]
* Increase Scroll Speed [RIGHT]
* Play/Pause [Spacebar]
* Full Screen [F11]
* Reset [CTRL]
